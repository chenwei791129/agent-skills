# 假別清單(時間類型)

SAP SuccessFactors「要求休假」表單「時間類型」combobox 的完整選項。選取時用
`fill` 填入**完整名稱(含中英文)**再按 Enter,名稱必須與下表完全一致。

| 正式名稱(填這個) | 常見口語 |
|---|---|
| `Annual Leave 公司特休假` | 特休、年假、特休假 |
| `Award Leave 獎勵假` | 獎勵假 |
| `Birthday Leave 生日假` | 生日假 |
| `Compassionate Leave 喪假` | 喪假 |
| `Compensated Rest Day 補休假` | 補休 |
| `Family Care Leave 家庭照顧假` | 家庭照顧假、家照假 |
| `Full-Paid Sick Leave 普通傷病假(全薪)` | 病假(全薪)、全薪病假 |
| `Half-Paid Sick Leave 普通傷病假(半薪)` | 病假(半薪)、半薪病假 |
| `Indigenous Ritual Leave 原住民祭儀假` | 祭儀假 |
| `Job Search Leave 謀職假` | 謀職假 |
| `Marriage Leave 婚假` | 婚假 |
| `Occupational Injury Sick Leave 公傷病假` | 公傷病假、職災假 |
| `Official Leave 公假` | 公假 |
| `Paternity Leave / Paternity Checkup Leave 陪產(檢)假` | 陪產假、陪產檢假 |
| `Personal Leave 事假` | 事假 |
| `Present without Access Card 出勤未刷卡` | 忘刷卡、未刷卡 |
| `Training Leave 培訓假` | 培訓假、訓練假 |
| `Unlimited Leave 無上限休假` | 無上限休假 |
| `Unpaid Military Service Leave 兵役留停` | 兵役留停 |
| `Unpaid Parental Leave 育嬰留職停薪假` | 育嬰假、育嬰留停 |
| `Volunteer Leave 公益假` | 公益假、志工假 |
| `Work From Home Permit 在家工作` | 在家工作、WFH、遠端 |

注意:

- 「一般病假」未指定全/半薪時,先向使用者確認要用全薪還是半薪。
- 病假類(普通傷病假全/半薪、公傷病假等)通常需附醫療證明,送出前依 SKILL.md
  步驟 3.5 上傳檔案。沒拿到檔案路徑時先向使用者確認是否要附。
- 口語對應不確定時,在表單按 `F4` 展開,以系統當下實際選項為準(清單可能隨公司
  政策調整)。
- 預設值為 `Annual Leave 公司特休假`。
